1) Запускаем сервер (по умолчанию указан порт 14649)
2) Файл базы данных SQLite Mobile.db по умолчанию создается в корне диска D. Путь к ней указан в классе BusinessLogic.Contexts.WalletContext.

3) Для создания пользователя можно воспользоваться POST запросом на адрес /User/CreateUser с JSON строкой вида { "Name": "имя_пользователя" }, получаем в ответ идентификатор нашего пользователя.

4) Далее можно воспользоваться следующими функциями:
4.1) Информация о балансе: GET запрос /Wallet/Balance?userId=идентификатор_пользователя

4.2) Пополнение счета: POST запрос /Wallet/Deposit с JSON строкой вида 
{
    "UserId": "идентификатор_пользователя",
    "Amount": сумма,
    "Currency": тип_валюты
}

4.3) Снятие со счета: POST запрос /Wallet/Withdraw с JSON строкой вида 
{
    "UserId": "идентификатор_пользователя",
    "Amount": сумма,
    "Currency": тип_валюты
}

4.4) Конвертация валюты: POST запрос /Wallet/Convert с JSON строкой вида 
{
    "UserId": "идентификатор_пользователя",
    "InitialAmount": сумма,
    "InitialCurrency": исходны_тип_валюты,
    "TargetCurrency": целевой_тип_валюты
}

5) Используемые типы валюты (указываются числом):
1 - доллары,
2 - евро,
3 - рубли
